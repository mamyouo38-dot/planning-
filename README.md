# 📈 Ma Progression — Conseil en Stratégie

Site interactif de suivi de progression personnel — L3 Éco-Gestion Finance → M1 Conseil en Stratégie.

## 🚀 Démarrage rapide

### Option 1 — GitHub Pages (recommandé)
1. Fork ce repo ou crée un nouveau repo
2. Upload le fichier `index.html`
3. Va dans **Settings → Pages → Source → main branch**
4. Ton site est en ligne sur `https://[ton-pseudo].github.io/[nom-du-repo]`

### Option 2 — En local
Ouvre simplement `index.html` dans ton navigateur. Aucune dépendance, aucune installation.

---

## ✨ Fonctionnalités

- **Dashboard** — Vue d'ensemble avec stats, alertes et heatmap d'activité
- **Planning global** — Timeline juillet 2025 → juin 2026
- **3 matières L3** — Politique monétaire, Finance de marché, Marchés financiers avec checklists de chapitres
- **Plan anglais** — 4 phases progressives avec programme semaine par semaine
- **Concours AST / IAE** — Suivi des deadlines et checklists de dossiers
- **Journal** — Notes quotidiennes catégorisées
- **Progression sauvegardée** — Tout est stocké dans le localStorage du navigateur

## 💾 Données

Toutes les données sont sauvegardées localement dans le navigateur (`localStorage`).  
Elles persistent entre les sessions mais restent sur ton appareil.  
Pour transférer d'un appareil à l'autre : copie le contenu de `localStorage.getItem('progression_v3')` et colle-le sur l'autre appareil via la console du navigateur.
